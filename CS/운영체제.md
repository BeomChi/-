#### 운영체제

운영체제는 하드웨어와 사용자(응용 프로그램) 사이에서 동작하는 시스템 소프트웨어로,

컴퓨터 자원을 효율적으로 관리하고 사용자가 컴퓨터를 편리하게 사용할 수 있도록 돕는다.



#### 운영체제가 하는 핵심 역할

* 자원관리

&nbsp;	cpu

&nbsp;	메모리

&nbsp;	저장장치

&nbsp;	입출력 장치



* 프로그램 실행 관리

&nbsp;	프로세스 생성 / 종료

&nbsp;	스케줄링

&nbsp;	스레드 관리



* 하드웨어 추상화

&nbsp;	복잡한 하드웨어를 직접 다루지 않아도 되게 함



* 사용자 편의성 제공

&nbsp;	파일 시스템

&nbsp;	UI

&nbsp;	시스템 호출(API)



결론

운영체제는 하드웨어 자원을 관리하고,

사용자와 응용 프로그램이 하드웨어를 효율적이고 안전하게 사용할 수 있도록

중개하는 시스템 소프트웨어이다.
