#### 람다

람다는 일회용 함수를 간결하게 작성하기 위한 문법으로,

C++에서는 익명 함수 객체를 생성한다.

기본 구조는 \[capture] (parameters) { function body }

* \[] : 킵처 리스트 (외부 변수 사용방식 지정)
* () : 매개변수 목록
* {} : 함수 본문



람다는 이름 없는 함수 객체를 생성한다

내부적으로는

struct \_\_Lambda

{

 	void operator()() const

 	{

 		....

 	}

};

와 같은 형태



캡처 방식

람다는 함수 외붕ㅔ 있는 변수를 캡처해서 사용할 수 있다.

* = : 외부 변수를 값으로 복사
* \& : 외부 변수를 참조로 캡처
* 특정 변수만 선택적으로 캡처 가능

캡처 방식에 따라 람다 내부에서 외부 변수의 수명과 변경 가능 여부가 달라진다.



캡쳐 방식			의미

\[=]				외부 변수를 값을 복사

\[\&]						     외부 변수를 참조로 캡처

\[a]						변수 a만 값 캡쳐

\[\&a]					변수 a만 참조 캡처

\[this]					현재 객체 캡처

\[=,\&a]					기본은 값,a만 참조
