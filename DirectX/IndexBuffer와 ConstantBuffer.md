#### Index Buffer

Index Buffer는 이미 생성된 정점들을 어떤 인덱스 순서로 정점을 참조하여 프리미티브를 구성할지를

정의하는 버퍼이다.

* 동일한 정점을 여러 삼각형에서 재사용 가능
* 버텍스 데이터 중복 제거
* 메모리 사용량 감소
* GPU의 Vertex Cache 효율 향상



중복되서 사용되는 정점 갯수를 줄이기 위해



#### Constant Buffer

Constant Buffer는 쉐이더에서 참조하는 변경 가능한 상수 데이터를 저장하는 버퍼이다.

* 변환 행렬(World / View / Projection)
* 카메라 위치
* 조명 정보
* 시간 값 등



정점 버퍼의 원본 데이터에는 영향을 주지 않고,

쉐이더에서 참조하는 값만 바꿔서 렌더링 결과를 동적으로 변화시킨다.

Constant Buffer는 DeviceContext를 통해 쉐이더 단계(VS, PS 등)에 바인딩된다.\\



상수 버퍼는 16바이트 정렬이 필수라는 물리적인 제약이 있다. 코드로 짤때 유의하자
